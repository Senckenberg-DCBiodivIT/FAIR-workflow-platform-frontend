This is a FAIR workflow platform which enables researchers to manage and execute scientific workflows on research datasets. The platform leverages on Workflow RO-Crates to encapsulate workflows along with their metadata and provenance information. These RO-Crates can be submitted, executed, and published all within one platform. The platform ensures that datasets and workflows adhere to FAIR principles, making them Findable, Accessible, Interoperable, and Reusable.

How to Use

  1. Browse Datasets: Visit the Datasets page to explore available research datasets. Click on any dataset to view detailed metadata, associated files, and workflow provenance information.
  2. Submit Workflows: To submit a computational workflow, you need to login with ORCID. Once authenticated, navigate to Submit Workflow and upload a workflow in RO-Crate format. Alternatively, you can select an existing Dataset and execute the corresponding workflow. You can monitor workflow status on the Workflows page.
  3. Retrieve RO-Crates: Datasets with their corresponding workflow are downloadable as RO-Crates, which capture complete workflow provenance. These packages contain workflow definitions, input parameters, output data, and execution metadata in a standardized format.
  4. FAIR Signposting: Dataset pages implement FAIR Signposting in their HTTP headers, enabling automated discovery of metadata and related resources by machines and indexing services.

Key Concepts & Documentation

  • FAIR Principles - Guidelines for scientific data management
  • RO-Crates - Lightweight method to package research data with metadata
  • Workflow RO-Crates - Profile for representing workflows and their results in RO-Crates
  • FAIR Signposting - Approach to make scholarly objects more accessible to machines

Source Code & Contact
